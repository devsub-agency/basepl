## Installation

You can add the pre-configured button to your project with the command below. If you only want to install the button block check the reference [here](https://basepl.com/docs/getting-started).

<CommandPreview
  npmCommand={'npx @basepl/cli add BaseplButton'}
  pnpmCommand={'npx @basepl/cli add BaseplButton'}
/>

Currently, basepl doesn't automatically update the payload types. So if you install the configuration and the component you have to update the types.
Simply run

<CommandPreview
  npmCommand={'npm generate:types'}
  pnpmCommand={'pnpm generate:types'}
/>

If not you will get an error in the BaseplButton/Component.tsx file. Because the BaseplButtonType is not present.

## Options

An overview of options you can set in the admin panel to customize the button.

### Link

- Label: The text that will be displayed on the button.
- Target Type: What type the button should be.
  - Page: Either a page reference or the slug to a page.
  - File Download: Link to a file download.
  - External URL: Link to an external website.
    - Open in new Tab: Should be self-explanatory.
    - No referrer: If a referral is present.
  - Email Address: mailto: link to an email address.

### Styling

- Icon: Check if you want to use an icon instead of the label text.
  - Icon: Choose the icon from the media collection.
  - Icon at Start: Choose if the icon should be placed at the start or the end of the button.
- Variant: Choose the style of the button. [Check out the examples](#example)
  - Default
  - Secondary
  - Outline
  - Destructive
  - Ghost
  - Link
- Size: Choose the size of the button.
  - Small
  - Medium
  - Large
