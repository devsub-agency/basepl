---
title: Link Field
description: The Link field allows you to add a link to your content.
---

## Installation

```bash
npx @basepl/cli add link
```

## Usage

```ts
import { Label } from '@/fields/link/config';
```

Simply add the label to the field array of your desired configuration.

```ts
fields: [
    link
]
```
