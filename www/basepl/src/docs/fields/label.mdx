## Installation

```bash
npx @basepl/cli add label
```

## Usage

```ts
import { Label } from '@/fields/label'
```

Simply add the label to the field array of your desired configuration.

```ts
fields: [label]
```

## Configuration

You can configure the max length of the label by passing it as a prop. The default length is 20.

```ts
fields: [label(100)]
```
