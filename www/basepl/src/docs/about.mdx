## Basepl

Basepl is an open source library built on top of [Payload CMS](https://payloadcms.com/) and [shadcn/ui](https://ui.shadcn.com/). It is designed to provide a set of pre-built templates, fields, blocks and components that you can easily use, customise and install from the command line.

<DocsCardsContainer />

## Support & Contribution

We welcome early feedback and ideas from the community to improve the library and plan the next steps. You are welcome to join our [Discord](https://discord.gg/b99KZMW2) or work with us on [GitHub](https://github.com/devsub-agency/basepl).

## Roadmap

There are a few things planned for 2025. The next things we want to provide out are the following:

1. **Globls**: Globals like header & footer.
2. **More Blocks & Components**: Tabs, Accordions, Cards etc.
3. **Templates**: Templates for forms, blogs, etc.
